# Sketch 001: Dashboard Layout & Navigation

## Design Question
What is the most effective navigation layout for PerGo's dual target audience (operators and makers), matching the clean, light-mode, monochromatic Notion-style aesthetic?

## How to View
open `.planning/sketches/001-dashboard-layout/index.html`

## Variants
- **A: Notion-Style Sidebar (Wide)** — A clean, classic left-hand sidebar containing workspace switcher, clear hubs (Overview, Connections, Playground, Logs, Settings) and a collapsible toggle. Excellent for clear visual organization and readability.
- **B: Slim Sidebar (Icons Only)** — Minimizes left navigation to a narrow icon-only bar with hover tooltips, maximizing horizontal canvas space. Perfect for high-density screens (like the developer playground).
- **C: Top Navigation (Horizontal)** — A traditional SaaS top-bar layout. Clears the left/right canvas entirely, giving a center-aligned look.

## What to Look For
- Test different viewport sizes (using the bottom-right viewport selector) to see how the layouts respond.
- Compare how accessible the menu hubs are between Variant A (text labels) and Variant B (icons only).
- Consider which layout scales best if we add more hubs later.
