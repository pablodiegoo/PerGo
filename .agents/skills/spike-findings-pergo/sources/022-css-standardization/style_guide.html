Biblioteca de componentes e diretrizes de estilo para o console da PerGo.

Style Guide

1. Estrutura de Títulos & Descrições

REGRA: Todos os títulos principais de páginas devem utilizar a mesma hierarquia (2xl/3xl font-bold) acompanhados de descrição curta (sm text-slate-500).

Título Principal da Página

Este é o texto de descrição que explica de forma objetiva o papel da tela.

Sub-seções de Conteúdo

Parágrafos de ajuda interna de cards ou áreas de configuração.

HTML 
<!-- Cabeçalho de Página Padrão -->
<div class="border-b border-zinc-200 pb-5 mb-8">
    <h1 class="text-2xl font-bold tracking-tight text-zinc-900">Título Principal</h1>
    <p class="text-zinc-500 text-sm mt-1">Descrição da página...</p>
</div>

<!-- Subtítulo Padrão -->
<h2 class="text-lg font-semibold text-zinc-900">Subtítulo da Área</h2>
<p class="text-zinc-500 text-xs mt-0.5">Descrição secundária...</p>

2. Paleta de Cores Coesa

REGRA: Evite cores primárias sólidas (azul puro, vermelho puro). Utilize a escala neutra do Zinc/Slate para o tema dark-light e cores dessaturadas para indicações.
Zinc 950 (Primary) #09090b
Blue 600 (Accent) #2563eb
Emerald 600 (Success) #059669
Amber 500 (Warning) #f59e0b

3. Sinalizações & Badges

REGRA: Status de envio/recebimento, origem ou tipo de agente devem usar badges leves de fundo claro com borda e texto colorido, evitando preenchimentos opacos escuros.
Sucesso / Ativo Conectado
Pendente / Alerta Pendente
Erro / Inativo Desconectado
Informativo API REST
HTML 
<!-- Badge de Sucesso -->
<span class="badge badge-sm bg-emerald-50 text-emerald-700 border-emerald-200 px-2 py-0.5 text-xs font-semibold">Sucesso</span>

<!-- Badge de Alerta -->
<span class="badge badge-sm bg-amber-50 text-amber-700 border-amber-200 px-2 py-0.5 text-xs font-semibold">Pendente</span>

<!-- Badge de Erro -->
<span class="badge badge-sm bg-rose-50 text-rose-700 border-rose-200 px-2 py-0.5 text-xs font-semibold">Erro</span>

<!-- Badge Informativo -->
<span class="badge badge-sm bg-blue-50 text-blue-700 border-blue-200 px-2 py-0.5 text-xs font-semibold">Info</span>

4. Formulários & Botões

REGRA: Rótulos superiores devem usar fonte xs semi-bold zinc-500 em caixa alta. Botões de ação primários devem usar o tema preto (bg-zinc-950) e secundários em outline leve.
HTML 
<!-- Campo de Entrada (Input/Select) -->
<div class="flex flex-col gap-1">
    <label class="text-xs font-semibold text-zinc-500 uppercase tracking-wider">Nome do Campo</label>
    <input type="text" class="form-input border border-zinc-300 rounded-md px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-zinc-900 focus:border-transparent transition-all" />
</div>

<!-- Botão Primário Preto -->
<button class="btn btn-black bg-zinc-950 hover:bg-zinc-900 text-white border-none font-semibold rounded-md px-4 py-2">
    Salvar
</button>

5. Tabelas de Informações

REGRA: Listas e relatórios tabulares devem usar a classe min-w-full com thead bg-zinc-50, dividindo as linhas suavemente via border-zinc-200 e aplicando hover.
Coluna 1 Coluna 2 Ação
Registro Exemplo A Valor exemplo associado
Registro Exemplo B Valor exemplo associado
HTML 
<!-- Tabela Consolidada PerGo -->
<div class="overflow-x-auto border border-zinc-200 rounded-lg shadow-sm">
    <table class="table min-w-full divide-y divide-zinc-200">
        <thead class="bg-zinc-50 text-zinc-500 text-xs font-semibold uppercase tracking-wider text-left border-b border-zinc-200">
            <tr>
                <th class="px-6 py-3">Título</th>
                <th class="px-6 py-3 text-right">Ações</th>
            </tr>
        </thead>
        <tbody class="divide-y divide-zinc-200 text-sm text-zinc-700 bg-white">
            <tr class="hover:bg-zinc-50/50 transition-colors">
                <td class="px-6 py-4">Dado</td>
                <td class="px-6 py-4 text-right">Botão</td>
            </tr>
        </tbody>
    </table>
</div>
