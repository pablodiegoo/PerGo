# Milestone v1.0 — Audit Report

## Gaps Analysis

### 1. Queue Depth counter leak (ingestion lock) (`QUEUE-02`)
* **Impact:** High. The ingestion API will permanently return HTTP 429 once a workspace reaches 1,000 total messages.
* **Remediation:** Decrement the QueueDepthTracker on message success or terminal fail in the outbound worker.

### 2. Plaintext whatsmeow device keys (`SEC-04`)
* **Impact:** High. whatsmeow keys are stored plaintext in PostgreSQL, posing security risks.
* **Remediation:** Implement a wrapper database store that encrypts key columns, or use pg_crypto triggers.

### 3. WABA Relative URL failure (`MEDIA-02`, `MEDIA-03`)
* **Impact:** Medium. Meta Cloud API cannot fetch media files from the relative proxy URL (/media/...) rewired internally.
* **Remediation:** Support an external host base URL configuration field to format full URL strings.

### 4. Fake Database Partitioning (`AUDIT-01`)
* **Impact:** Medium. Declared schema does not support declarative partitioning. All inserts go into the parent `audit_logs` table.
* **Remediation:** Re-migrate `audit_logs` using `PARTITION BY RANGE (created_at)` and attach monthly tables using `PARTITION OF`.

### 5. ClientOutdated version refresh (`WAWEB-06`)
* **Impact:** Low. connection fails on outdated events.
* **Remediation:** Execute SetWAVersion on Outdated events.

### 6. Terminal status missing on LoggedOut (`WAWEB-07`)
* **Impact:** Low. Logged-out devices trigger reconnection loops on startup.
* **Remediation:** Set status to `terminal` in the DB when LoggedOut occurs.
