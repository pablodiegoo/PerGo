# Plan: Amazon SES, Mautic, Tracking Engine & Email Webhooks

## Verification Criteria
- `tracking.go` injects 1x1 GIF open-tracking pixel and rewrites `<a href>` links with HMAC signature when enabled.
- `ses.go` implements `SESProvider` for Amazon SES API dispatching.
- `mautic.go` implements `MauticProvider` for Mautic REST API dispatching.
- `internal/inbound/email.go` handles click/open tracking endpoints and normalizes SES SNS & Mautic webhook events.
- All unit tests in `internal/channel/email/...` and `internal/inbound/...` pass.

## must_haves
- Open tracking pixel MUST be injected right before `</body>` or appended if `</body>` is absent.
- Click tracking URLs MUST be HMAC-signed to prevent URL tampering.
- Webhook handlers MUST convert SES/Mautic events into standard PerGo message status updates.
