- Run `go test -v ./internal/api/handler/admin -run TestInbox` to verify that 24h window checks and template publishers are validated.

### Manual Verification
1. Run PerGo.
2. Open the Inbox interface.
3. Select a conversation using a WABA channel:
   - If the last inbound message is older than 24h, verify the warning banner displays and textarea is disabled.
   - Click the "Templates" button, choose a template (e.g. `welcome_optin`), populate variables, and send. Verify that the window is successfully reopened upon simulated reply.
4. Click "Novo Chat". Choose WABA channel and verify free-text is disabled. Choose Telegram and verify free-text is enabled.
