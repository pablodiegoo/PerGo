       - `Encryptor` struct holding the KEK (32-byte key from env var or file)
       - `NewEncryptor(kek []byte) *Encryptor` — validates KEK is 32 bytes
       - `Encrypt(plaintext []byte) (ciphertext []byte, keyID string, keyVersion int, err error)` — generates random DEK (32 bytes), wraps DEK with KEK using AES-256-GCM (fresh 12-byte nonce per Seal), encrypts plaintext with DEK, returns envelope: nonce + encrypted_DEK_len + encrypted_DEK + encrypted_plaintext. key_id = "default", key_version = 1
       - `Decrypt(ciphertext []byte) (plaintext []byte, err error)` — parses envelope, unwraps DEK with KEK, decrypts plaintext
       - Use `crypto/aes`, `crypto/cipher`, `crypto/rand` from stdlib

       - `WorkspaceIDFrom(ctx context.Context) (uuid.UUID, bool)` — retrieves workspace_id, returns false if missing
       - `RequireWorkspaceID(ctx context.Context) (uuid.UUID, error)` — calls WorkspaceIDFrom, returns error if missing

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → Auth middleware | Untrusted API key crosses into authentication layer |
| Auth middleware → API key cache | Cached keys must be invalidated on revocation |
| Crypto subsystem → PostgreSQL | Encrypted credentials stored at rest |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Tampering | API key brute-force | high | mitigate | SHA-256 hashing makes offline brute-force expensive; rate limiting deferred to Phase 3 |
| T-02-02 | Information Disclosure | API key in Authorization header | medium | mitigate | TLS required for production; key is one-way hashed, not reversible |
| T-02-03 | Elevation of Privilege | Cross-tenant API key usage | high | mitigate | Auth middleware injects workspace_id into context; all subsequent queries scope by workspace_id |
| T-02-04 | Tampering | Nonce reuse in AES-GCM | critical | mitigate | Fresh 12-byte nonce from crypto/rand per Seal call; nonce prepended to ciphertext |
| T-02-05 | Information Disclosure | Cache stale entry after revocation | medium | mitigate | Revoke() invalidates cache entry immediately; TTL = 5 minutes max staleness |
| T-02-06 | Tampering | HMAC comparison timing attack | high | mitigate | Use hmac.Equal() for constant-time hash comparison |
| T-02-SC | Tampering | Go module dependencies | high | mitigate | All packages are well-established Go ecosystem libraries (crypto/aes, crypto/sha256 are stdlib) |
</threat_model>
