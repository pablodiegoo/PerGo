<objective>
Resolve all 6 audit gaps identified in the v1.0 milestone audit report:
1. Re-migrate audit_logs to use declarative RANGE partitioning.
2. Encrypt whatsmeow device keys at rest in PostgreSQL via views and triggers.
3. Decrement QueueDepthTracker on successful, expired, or terminally failed message resolution.
4. Prepend external host base URL to relative media proxy URLs in WABA adapter.
5. Handle ClientOutdated events by refreshing the WA version and reconnecting.
6. Mark device status terminal on LoggedOut event, and prevent status update to disconnected on exit.
</objective>
