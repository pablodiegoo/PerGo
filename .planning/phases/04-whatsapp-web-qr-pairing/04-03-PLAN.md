<task type="auto" tdd="true">
  <name>Task 1: QR pairing flow (RED + GREEN)</name>

  <description>
  Build the QR code pairing mechanism:
  - SessionManager.StartPairing(workspaceID, phone) creates a new whatsmeow device and returns a QR channel
  - whatsmeow's QR code event handler sends QR code PNG data (base64) through the channel
  - QR codes expire after ~20s — handler detects expiry and sends new QR
  - Pairing completes when phone scans QR — handler sends success signal through channel
  - On pairing success: save device to DB, add to session registry, start session goroutine
  - On pairing failure/timeout: clean up device, return error
  </description>

  <deviations>
  - Use channel-based flow: manager returns <-chan QRPairingEvent
  - QRPairingEvent has Type (qr_code, paired, error, expired) and Data ([]byte for image)
  - whatsmeow provides QR as image/png — encode as base64 for embedding in HTML img src
  - Timeout after 5 minutes of no scan — clean up
  </deviations>

  <files_modified>

    - internal/session/manager.go
  </files_modified>

  <success_criteria>
