# Patterns for Phase 24.2

## Files to Modify

### 1. `internal/integration/typebot/forwarder.go`
- **Role:** Modifies the Typebot forwarder to accurately populate missing routing and correlation fields in outbound queue messages.
- **Data Flow:** Maps `event.ConnectionID` to `outMsg.ConnectionID`, `event.To` to `outMsg.SenderIdentity`, and a newly generated `traceID` to `outMsg.TraceID` before serializing and publishing to the `messages.outbound` queue.
- **Closest Analog/Context:** Existing logic inside `SyncInboundMessage` where `domain.QueueMessage` is constructed and sent using the `f.publisher.Publish` method.
- **Code Snippet:**
  ```go
  	// 5. Publish returned messages from Typebot to messages.outbound
  	for _, m := range messages {
  		traceID := uuid.New().String()
  		// Create an outbound QueueMessage
  		outMsg := domain.QueueMessage{
  			WorkspaceID:    event.WorkspaceID,
  			ConnectionID:   event.ConnectionID,
  			SenderIdentity: event.To,
  			TraceID:        traceID,
  			Channel:        event.Channel,
  			To:             event.From,
  			Body:           extractText(m),
  		}

  		// Ensure body is not empty before sending
  		if outMsg.Body != "" {
  			b, err := json.Marshal(outMsg)
  			if err != nil {
  				slog.Error("failed to marshal outbound message from typebot", "error", err)
  				continue
  			}

  			err = f.publisher.Publish(ctx, "messages.outbound", b, traceID)
  			if err != nil {
  				slog.Error("failed to publish outbound message from typebot", "error", err)
  			}
  		}
  	}
  ```

### 2. `internal/integration/typebot/forwarder_test.go`
- **Role:** Verifies that `SyncInboundMessage` accurately populates `ConnectionID`, `SenderIdentity`, and `TraceID` when publishing outbound replies.
- **Data Flow:** Modifies the `mockPublisher` to capture the published `[]byte` data and `traceID`, then adds a test case to unmarshal the payload into `domain.QueueMessage` and assert that the three new fields match the inbound event attributes and the generated trace ID.
- **Closest Analog/Context:** Existing `mockPublisher` and `TestTypebotForwarder` test suite.
- **Code Snippet:**
  ```go
  type mockPublisher struct {
  	called   bool
  	data     []byte
  	traceID  string
  }

  func (m *mockPublisher) Publish(ctx context.Context, subject string, data []byte, traceID string) error {
  	m.called = true
  	m.data = data
  	m.traceID = traceID
  	return nil
  }
  ```
