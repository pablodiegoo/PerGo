# Plan: Deepen Media Engine to Consolidate Storage Pipelines

## Objective
Refactor the Media Engine to be a deep module. Consolidate hashing, content-type sniffing, S3 key construction, and proxy URL formatting within the Media Engine, simplifying the outbound and inbound processors and removing duplicated storage coordination logic.
